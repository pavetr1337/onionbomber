# Onion Bomber
Base for SMS bomber in Telegram

## Installation
1. Clone/Download this repo
2. pip install -r requirements.txt
3. Create .env file:
```
BOT_TOKEN = "TOKEN_FROM_BOTFATHER"
PROXYLEET_KEY = "KEY_FROM_PROXYLEET"
```
